# Quick Start Guide - Mikilele Events Admin

## 5-Minute Setup

### Step 1: Access Admin Dashboard
Visit: https://mikilele-customer.vercel.app/admin

### Step 2: Create Your First Event
1. Click "Events Admin"
2. Create event
3. Add ticket types
4. Publish

### Step 3: Create Your First Course
1. Click "School Admin" → "Courses"
2. Click "Create Course"
3. Enter title and goals
4. Click "Generate Curriculum with AI"
5. Review and save

### Step 4: Launch Your First Campaign
1. Click "Marketing Campaigns"
2. Create campaign
3. Add email step
4. Create audience segment
5. Link segment to campaign
6. Activate campaign

### Step 5: Enroll Your First Student
1. School Admin → Enrollments
2. Click "Enroll Student"
3. Select course and student
4. Set payment details
5. Save
